# 忘却前夜 Wiki 视频托管仓库

本仓库用于托管 [忘却前夜中文维基](https://morimens.huijiwiki.com/) 所需的视频媒体资源。


## 使用方法

在 Wiki 页面中，使用以下 `iframe` 代码即可调用本仓库的视频播放器。只需修改 URL 最后的 `?v=视频文件名.mp4` 即可实现视频切换。

<iframe src="[https://你的用户名.github.io/你的仓库名/index.html?v=pv_first_chapter.mp4](https://你的用户名.github.io/你的仓库名/index.html?v=pv_first_chapter.mp4)" width="100%" height="450px" frameborder="0" allowfullscreen></iframe>

当前视频索引
* pv_first_chapter.mp4

（后续新增视频可直接在此追加，便于统一管理）
